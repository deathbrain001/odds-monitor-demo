
# Odds Monitor Demo (Ready to Deploy)

## What is this?
A simple, read-only odds monitoring website using a free odds API.
- Auto refresh every 10 seconds
- Color alerts: Green / Yellow / Red
- No betting, no changes (safe demo)

## Setup
1. Install Python 3.9+
2. Get a free API key from https://the-odds-api.com/
3. Copy `.env.example` to `.env` and paste your API key
4. Install deps:
   pip install -r requirements.txt

## Run
python app.py

Open in browser:
http://localhost:5000
